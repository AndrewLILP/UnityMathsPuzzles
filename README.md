# Turing Test CSGD 2025

My Unity game project for the Circuit Stream Game Dev Bootcamp.

## Features
- Puzzles based on the 7 Bridges of Konigsberg Mathematics Problem

## Development Setup
- Unity 6 (39f1)
- Built with Universal Render Pipeline (URP)
